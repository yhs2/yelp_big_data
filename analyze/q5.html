For predicting stars, we use attributes such as states, price range, and noise level to do prediction on stars. As shown in the figure, there is correlation between true stars and predicted stars. We have a validation r2 score of 0.5, which is not a high score. However, it is a meaningful result, because we found that some features can affect the prediction in the right way. One main problem with the Yelp dataset is that lots of businesses do not have attributes such as wifi, noise level, which makes our prediction dataset small. If these kinds of attributes can be filled up. We can collect more data and add more features, which can make our predictions much better.
